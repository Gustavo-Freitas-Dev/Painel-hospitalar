# 🏥 Hospital Queue API

API backend desenvolvida com **FastAPI** para gerenciamento de filas de atendimento em hospitais ou clínicas.

O sistema permite cadastrar pacientes, gerar senhas de atendimento e visualizar a fila de espera.

---

## 🚀 Funcionalidades

- Cadastro de pacientes
- Geração automática de senhas de atendimento
- Visualização da fila de espera

---

## 🛠️ Tecnologias utilizadas

- Python
- FastAPI
- Uvicorn

---

## 📚 Objetivo do projeto

Este projeto foi desenvolvido para praticar conceitos de **desenvolvimento backend**, como:

- criação de APIs REST
- organização de rotas
- gerenciamento de filas
- estruturação de projetos com FastAPI
